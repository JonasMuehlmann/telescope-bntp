# telescope-bntp
telescope.nvim finders for https://github.com/JonasMuehlmann/bntp.go

# Setup

Add the following line somewhere after `require('telescope').setup()`.
```lua
require('telescope').load_extension('bntp')
```

## Available commands

```vim
:Telescope bntp documents
```
